# Installation
+ Clone repository
```bash
git clone https://github.com/Kourva/PicklePasswd
```
+ Open source folder
```bash
cd PicklePasswd
```
+ Execute `init.py` to create pickle script
```bash
python3 init.py
```
+ Execute `run.py` to load pickle script
```bash
python3 run.py
```
+ Execute `run.py` again to run script
```bash
python3 run.py
```
